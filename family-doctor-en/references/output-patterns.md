# Output Patterns

Use English by default. Keep answers short, practical, and safety-forward.

## Symptom Triage Template

```
First, safety: based on what you described, {red flag status/main risk to rule out}.
Special-risk status: {child/older adult/pregnancy/allergy history/chronic disease/immunocompromised/anticoagulant/no special high-risk factor known}.

What to do now:
1. {immediate action}
2. {home care}
3. {recheck timing}

Avoid:
- {unsafe or unhelpful action}

Seek urgent care / ER / emergency services if any of these happen:
- {red flag 1}
- {red flag 2}
- {red flag 3}

I still need to confirm: {1-3 key questions}
```

## Injury Template

```
Handle the most dangerous issue first: {bleeding/cooling/immobilizing/airway/poisoning exposure}.

Steps:
1. {first step}
2. {second step}
3. {third step}

Get medical care if:
- {threshold}

If you can, tell me: {age, injury time, body part, severity, medical conditions/medicines}, and I can help sort home care vs urgent care vs ER.
```

## Emergency Wording

```
This is not a good situation to watch at home. Please call emergency services now. If you are in the US or Canada, call 911; in the UK or Ireland, call 999 or 112; in Australia, call 000. While waiting, {safe position/continuous pressure/no food or drink/bring medication bottle}.
```

## Special Group Wording

```
Because this involves {a child/an older adult/pregnancy/allergy history/chronic disease/immune suppression/anticoagulants}, the threshold for getting medical care should be lower than for a healthy adult. If {key red flags} appear, seek same-day care or emergency care.
```
