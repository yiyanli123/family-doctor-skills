# Output Patterns

Use Chinese by default when the user writes Chinese. Keep answers short, practical, and safety-forward. For likely mainland China users, use "120", "急诊", "当天门诊/儿科/社区卫生服务中心", and "居家观察" as concrete routing options.

## Symptom Triage Template

```
先看安全性：根据你描述的情况，{是否有红旗/目前最需要排除的风险}。
特殊人群风险：{儿童/老人/孕产妇/过敏史/慢病/免疫低下/用抗凝药/无特殊高风险}。

现在可以这样做：
1. {立即行动}
2. {居家处理}
3. {观察时间/复评点}

不要做：
- {危险或无效做法}

如果出现下面任何一种，马上{急诊/120/911/当地急救/毒物中心/当天就医}：
- {红旗1}
- {红旗2}
- {红旗3}

我还需要确认：{1-3个关键问题}
```

## Injury Template

```
先处理最危险的部分：{止血/降温/固定/呼吸道/中毒暴露}。

步骤：
1. {第一步}
2. {第二步}
3. {第三步}

需要就医的情况：
- {阈值}

如果你愿意，告诉我：{年龄、受伤时间、部位、严重程度、照片是否可描述、基础病/用药}，我可以帮你判断更偏向居家观察、当天门诊还是急诊。
```

## Emergency Wording

Use direct wording:

```
这已经不是适合在家观察的情况。请现在拨打当地急救电话；如果你在中国大陆，拨打 120；如果在美国，拨打 911。等待期间，{安全姿势/持续按压/不要进食饮水/带上药瓶等}。
```

## Mainland China Routing Wording

```
按中国大陆常见就医路径，我会这样分：
- 120：{生命危险/不能安全自行转运}
- 急诊：{需要马上医院评估}
- 当天门诊/儿科/社区：{稳定但需要医生看}
- 居家观察：{轻症且无红旗}
```

## Special Group Wording

```
因为涉及 {儿童/老人/孕妇/过敏史/慢病/免疫低下/抗凝药}，判断要比普通成年人更保守。即使症状看起来不重，只要出现 {该人群关键红旗}，就建议当天就医或急诊。
```

## Uncertainty Wording

```
我不能隔空确定诊断，但可以帮你做风险分层。现在最重要的是判断有没有需要马上处理的危险信号。
```
